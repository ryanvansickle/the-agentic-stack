# The Agentic Stack

**A Metaframework for Autonomous Agent Systems**

The first comprehensive architectural map of the agentic AI landscape. 9 layers, 104+ primitives, 5 planes, 12 design patterns.

Version 2.0 · March 2026

## About

The Agentic Stack is a vendor-neutral metaframework that names, classifies, and positions every functional primitive required to build, deploy, govern, and evolve autonomous agents. It is not a product or a specification — it is a shared language for an industry that needs one.

## Structure

- **9 Layers** (L0–L8): From infrastructure substrate to agent economics
- **5 Planes**: Cross-cutting concerns that propagate through every layer
- **12 Design Patterns**: Canonical architectures for production agent systems
- **The Learning Engine**: Six timescales of agent improvement
- **The Compounding Agent**: The framework's culminating thesis
